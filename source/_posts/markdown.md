## 锚点
### 锚点的原理
锚点本质上是通过html的超链接，通过dom节点的id以及a标签的超链接去进行关联。
```html
    # 标题
    // 上面的markdown会被转换为如下
    <h1 id="标题">
        标题
    </h1>
```
```html
    [跳转到标题](#标题)
    // 通过链接语句相当于html的如下
    <p>
        <a href="#标题">
            跳转到标题
        </a>
    </p>
```
